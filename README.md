# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.



# 安装
1.git clone --depth 1 https://github.com/Knowckx/viteinit.git appName
2.删掉 .git文件夹
3.pnpm install



# Command

pnpm dev


# 打包
替换 viteinit

pnpm run build

pnpm preview --host

直接打开APP，刷新页面进行更新
