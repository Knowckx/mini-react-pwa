# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.



# 安装
git clone --depth 1 https://github.com/Knowckx/viteinit.git appName

删掉 .git文件夹

pnpm install

替换 viteinit

# Command

pnpm dev

pnpm run build

pnpm preview --host

直接打开APP，刷新页面进行更新
